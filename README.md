# üì∏ Google Photo Gallery Clone (Full-Stack PHP)

A dynamic, database-driven web application inspired by Google Photos. This project demonstrates the integration of a front-end UI with a **MySQL** backend, featuring user authentication and a responsive image gallery.

---

## üöÄ Key Features

* **Dynamic Image Gallery**: Organizes photos into categories like **Plants**, **Animals**, and **Historical Places**.
* **User Authentication**: A complete **Sign Up** and **Login** system that validates user credentials against a MySQL database.
* **Interactive Contact Form**: A functional "Contact Us" section where user messages, names, and emails are captured and stored.
* **Database Integration**: Managed via **phpMyAdmin**, featuring dedicated tables for `signup` and `contact` information.
* **Responsive Design**: A clean, modern UI with a navigation bar and hover effects for an intuitive user experience.

---

## üõ†Ô∏è Technical Stack

* **Front-end**: HTML5, CSS3 (Custom styling for the dark theme and gallery layouts).
* **Back-end**: **PHP** (Server-side logic for form handling and database connectivity).
* **Database**: **MySQL** (Relational database for storing user accounts and feedback).

---

## üìÇ Project Structure

- `index.php`: The landing page with a modern hero section.
- `gallery.html`: The main hub for navigating different photo collections.
- `signup.php` / `login.php`: Backend logic for managing user access.
- `contact.php`: Form handling for user feedback and inquiries.
- `Database/`: Contains the SQL scripts to recreate the `first_attempt` database.

---

## üíª How to Run (Local)

1.  **Start XAMPP**: Ensure the **Apache** and **MySQL** modules are running in your XAMPP Control Panel.
2.  **Clone/Move the Project**: Place the project folder in your `C:/xampp/htdocs/` directory.
3.  **Setup Database**:
    - Open `localhost/phpmyadmin`.
    - Create a database named `first_attempt`.
    - Import the provided `.sql` files to create the `signup` and `connect` tables.
4.  **Access the App**: Open your browser and go to `localhost/your_folder_name/index.php`.
